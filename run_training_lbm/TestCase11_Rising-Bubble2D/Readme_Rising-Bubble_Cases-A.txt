SUMMARY

Test cases of rising bubble inspired from Ref [1, Section III.D]. The density
ratio is 829 and the interface width is set equal to 8. In order to keep small
the Cahn number (Ca=W/L=0.02), the domain has been extended up to L=400 and the
bubble radius has been set to R=37.5.

AUTHOR
A. CARTALADE

COMMENT

The test cases A2 & A5 have run on Topaze A100 in respectively 13min and 39min
(see output file .e in each directory).
Test case A4 has failed (NaN after 150.000 time it)
